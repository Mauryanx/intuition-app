# Intuition App - Product Backlog

## App Vision
A React Native brain training app focused on developing intuitive abilities through engaging minigames. The app addresses the critical need for human intuition in the AI age, helping users develop "gut feelings" and subconscious processing skills through real-world application scenarios.

## Core Features
- **Powerful Onboarding Funnel**: Explains why intuition matters, current pain points, and future benefits
- **Individual Minigames**: Target different aspects of intuition
- **Progress Tracking & Personalization**: Adaptive learning paths
- **Social Features**: Leaderboards and competitive elements
- **Subscription Model**: Superwall integration post-onboarding

## Priority Levels
- **P0 (Critical)**: Core functionality, onboarding, basic minigames, user authentication
- **P1 (High)**: Progress tracking, social features, subscription, real-world application
- **P2 (Medium)**: Advanced minigames, analytics, specialized training, gamification
- **P3 (Low)**: Nice-to-have features, advanced AI integration, experimental features

## Dependencies
- Onboarding must be completed before subscription integration
- Basic minigames must be implemented before advanced analytics
- User authentication must be in place before social features
- Core progress tracking must be implemented before personalization features
- Basic minigames must be completed before specialized training
- Social features must be implemented before advanced gamification
